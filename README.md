AI Powered Personalized Learning Path Generator

 Description
This project generates a personalized learning path for users based on their skill level and learning goals. It helps learners follow a structured roadmap to improve their knowledge efficiently.

Features
- Accepts user skill level (Beginner, Intermediate, Advanced)
- Accepts learning goal (Python, Java, etc.)
- Generates a customized learning path
- Simple and user-friendly interface

Technologies Used
- Python
- Basic AI logic

How to Run
1. Clone the repository
2. Run the Python file

python learning_path_generator.py
 Example
Input:
Skill Level: Beginner  
Goal: Python  

Output:
- Learn Python Basics  
- Variables and Data Types  
- Control Statements  
- Functions  
- Basic Projects
